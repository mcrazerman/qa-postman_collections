Postman-коллекция для [https://restful-booker.herokuapp.com/apidoc/index.html](https://restful-booker.herokuapp.com/apidoc/index.html).
Проект представляет собой тренировчное API имитирующее сервис по бронированию номера в отеле.
Коллекция запускается в порядке, который Postman задаст по умолчанию.
