Коллекция для публичного проекта [https://fakeapi.platzi.com/](https://fakeapi.platzi.com/). Проект представляет собой прототип для создания API для онлайн магазина.
Есть [Swagger](https://api.escuelajs.co/docs#/) и [Документация](https://fakeapi.platzi.com/en/about/introduction/).

В целях демонстрации реализована только часть endpoint-ов, а именно:
- Позитивные кейсы:
  - Создание нового товара
  - Получение товара по id
  - Фильтрация товаров
  - Удаление товара
- Негативные кейсы:
  - Ошибка создания товара

Негативные кейсы можно запускать в любом порядке.
Позитивные кейсы нужно запускать в указанном в readme-файле порядке (чтобы все авто-тесты были "зелеными")
